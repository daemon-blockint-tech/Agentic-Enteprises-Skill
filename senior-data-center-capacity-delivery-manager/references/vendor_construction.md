# Vendor and Construction Management

## Vendor roster

| Vendor | Scope | Contract type | DRI |
|---|---|---|---|
| Colo / landlord | Space, power, SLA | Lease / order |
| GC | Fit-out | Lump sum / GMP |
| MEP subs | Trade work | Subcontract |
| Commissioning | IST scripts | T&M or fixed |
| Carriers | Fiber | IRU / cross-connect |

## Site cadence

- **Daily** huddle during peak construction (15 min)
- **Weekly** owner/GC schedule review
- **Monthly** executive walk (safety + quality)

## RFIs and submittals

| Metric | Target |
|---|---|
| RFI response | <5 business days (design lead) |
| Submittal approval | Per contract; track aging |

Aging RFIs on critical path → escalate same day.

## Change orders

Log: description, $, schedule days, approver, link to ECO.

No work without approved CO on critical path items.

## Safety and access

- Site induction for all visitors
- LOTO procedures for energization events
- After-hours work permits

## Quality and punch list

- Punch by zone; target zero critical before G3
- Photo evidence for hidden work before close-up

## Acceptance documentation

Package for gate:

- Test reports, warranties, O&M manuals
- Training for facilities operators
- Spares list
