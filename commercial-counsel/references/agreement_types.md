## Common stack

| Document | Typical purpose |
|---|---|
| MSA / framework | Master legal terms |
| Order form / subscription | Commercial terms, SKUs, fees |
| SOW / statement of work | Scoped services, milestones |
| DPA / BAA | Personal data processing |
| SLA exhibit | Uptime, credits, support tiers |
| Security exhibit | Technical and audit rights |

Later documents usually control on conflict only if stated—verify order of precedence clause.

## Review order

1. Read precedence and entire agreement clause
2. MSA baseline
3. Order form commercial terms
4. DPA if personal data
5. SOW for deliverables and acceptance
6. Exhibits last (often most negotiated)
