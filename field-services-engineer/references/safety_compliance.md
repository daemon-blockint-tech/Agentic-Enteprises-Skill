# Safety and Compliance

## Universal rules

- Follow **site safety briefing** and local regulations
- **PPE**: steel toe, gloves, eye protection; arc gear if panel work
- **Two-person lift** above site weight limit
- **No lone work** in energized electrical rooms if site forbids
- Stop work if unsafe; escalate before continuing

## Electrical

| Activity | Requirement |
|----------|-------------|
| Rack PDU connect | De-energized ports only if hot work prohibited |
| Panel/breaker work | Qualified electrician or LOTO certified only |
| UPS/battery | Site-specific training; no metal tools near bus |

**Lockout/tagout (LOTO):**

1. Notify facilities and change manager
2. Identify isolation point
3. Lock, tag, verify zero energy
4. Work
5. Remove LOTO in reverse order with verification

## Ladder and lift

- Use colo lift booking; certified operator if required
- No rack work from unstable ladder
- Overhead clearance for lifting gear

## ESD

- Wrist strap in staging areas when required
- Servers on anti-static mats
- Handle drives and DIMMs per vendor static rules

## Customer and data

- No logical access to customer data on disks
- Do not photograph screens with credentials
- Escort required in secure areas; sign NDAs on file

## Hazards

| Hazard | Mitigation |
|--------|------------|
| Hot aisle | Limit time; hydration |
| Fiber laser | Never look into port; caps on |
| Sharp rack edges | Gloves |
| Heavy doors | Buddy system |

## Incident on site

1. Protect people first
2. Notify site security and internal manager
3. Document facts; no blame assignment in field notes
4. Medical: call local emergency per site procedure

## Compliance records

- Training certs current (electrical, lift, site-specific)
- Insurance / COI for vendor smart-hands
- Change ticket retained for audit

## When to refuse work

- No approved change
- Missing PPE or escort in secure zone
- Design conflict creating electrical overload risk
- Pressure to bypass safety for schedule

Document refusal and escalate to delivery manager.
