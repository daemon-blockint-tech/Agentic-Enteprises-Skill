# Customer Success

## Onboarding Playbooks

### SaaS Onboarding (30-60-90 Day)

**Day 1-7: Setup & Foundation**
- Welcome sequence (email + in-app)
- Account configuration wizard
- Team member invitations
- Data import / integration setup
- First login tutorial

**Day 8-30: First Value**
- Use case-specific walkthrough
- Configure first workflow/report
- Connect integrations
- Attend live training or watch recorded
- Complete "first success" milestone

**Day 31-60: Adoption**
- Advanced feature introduction
- Best practices session
- Peer/customer reference call
- Usage review and optimization
- Expand to additional teams (if applicable)

**Day 61-90: Optimization**
- Quarterly business review prep
- ROI documentation
- Renewal conversation (if annual)
- Advocate identification
- Expansion opportunity assessment

## Health Scoring

### Quantitative Metrics

| Metric | Source | Calculation |
|---|---|---|
| Login frequency | Product analytics | Days with session / total days |
| Feature breadth | Product analytics | % of available features used |
| API calls | Product analytics | Weekly volume trend |
| Data volume | Product analytics | Records/projects created |
| NPS | Survey | Promoter % - Detractor % |
| CSAT | Support tickets | Avg score post-resolution |
| Support volume | Helpdesk | Tickets per month |
| Time to value | Onboarding data | Days to first success |

### Qualitative Signals
- Champion changes jobs (risk)
- Champion promoted (opportunity)
- M&A activity (uncertainty)
- Budget freeze announced (risk)
- New competitive evaluation (risk)
- Positive social mentions (opportunity)

### Score Calculation
```
Health Score = weighted_average(usage, adoption, sentiment, engagement)

Green (80-100): Expand and grow
Yellow (50-79): Monitor and intervene
Red (0-49): Escalate and save
```

## Renewal Management

### Renewal Timeline (Annual Contracts)

| Phase | Timing | Activities |
|---|---|---|
| Early warning | T-180 days | Health score review, risk flag |
| Value review | T-120 days | ROI documentation, usage reports |
| Initial conversation | T-90 days | Intent check, budget confirmation |
| Negotiation | T-60 days | Pricing, terms, expansion |
| Close | T-30 days | Signature, procurement |
| Handoff | T-0 | Renewed or churned |

### Risk Indicators
- Health score dropped to red
- No response to outreach (3+ attempts)
- Champion left company
- Usage declining for 60+ days
- Support tickets with negative sentiment
- Competitive mentions

### Save Play
1. Executive business review with decision maker
2. Address specific pain points with roadmap
3. Offer temporary concession (discount, free services)
4. Escalate to leadership if strategic account
5. Document learnings for product/CS team

## Expansion Playbooks

### Land-and-Expand Triggers
- Usage approaching plan limits
- New team requesting access
- Multiple departments using informally
- Champion advocating internally
- Positive ROI demonstrated

### Expansion Conversation
```
"Based on your usage, you've grown [X%] since onboarding.
You're currently on [Plan] with [Usage].

I see an opportunity to:
- [Specific expansion play 1]
- [Specific expansion play 2]

Would a brief call to explore make sense?"
```

## Customer Advocacy

### Advocate Program
- Identify promoters (NPS 9-10)
- Offer incentives: swag, event access, exclusive features
- Request: case study, reference call, testimonial, review
- Maintain relationship quarterly

### Reference Program
- Track willingness to be a reference
- Match prospects to references by use case
- Compensate or recognize references
- Keep reference pool fresh (3-5 per segment)
