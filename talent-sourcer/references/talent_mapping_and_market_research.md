## When to build a talent map

Build a map when:

- New role family or geo with unknown pool
- Executive or scarce skill (low boolean yield)
- Build-vs-buy team stand-up (hire 5–20 in quarter)
- Competitive poach campaign with named target companies
- HM requests "who exists at X?"

Skip heavy mapping for high-volume evergreen reqs with proven booleans.

## Map structure

Layers (bottom to top):

1. **Market** — segment, demand drivers, compensation pressure
2. **Companies** — tiers: primary targets, secondary, avoid
3. **Teams** — org units known to house ICP
4. **Individuals** — named profiles (where appropriate and lawful)
5. **Relationships** — referrals, alumni, community ties

## Company and competitor analysis

For each target company capture:

| Field | Notes |
|---|---|
| Size / stage | Startup vs enterprise affects pitch |
| Tech stack | Overlap with your stack |
| Recent news | Layoffs (inflow), funding (retention risk) |
| Open roles | Competing reqs on market |
| Poachability | Brand, equity, visa, culture |
| Restrictions | Partners, NDAs, customer no-poach |

**Talent pool analysis** = aggregate patterns, not dossiers on every employee.

## Market sizing

Rough funnel for planning:

```
Addressable titles in geo
  × % with must-have skills (estimate from sample searches)
  × % likely reachable (not founder-only, etc.)
  = actionable pool
```

Compare to **hiring target** and timeline. If pool < 3× target, widen ICP or geo.

## Warm path identification

| Path | How to use |
|---|---|
| Employee referrals | Ask team for intros before cold |
| Alumni | Same-school or ex-company ties |
| OSS / community | Maintainer relationships |
| Investors / advisors | Intro requests with approval |
| Conference | Follow-up within 48h of talk |

Log warm path on CRM contact for higher priority outreach.

- Pool estimate: N qualified in geo
- Top 5 target companies: ...
- Key risk: [competition / visa / remote policy]

## Company tiers

|------|-----------|-----------|
| A | | |
| B | | |

## Named targets (A-tier sample)

|------|---------|-------|------|-------|

## Warm paths
- ...

## Gaps

- Skills in short supply

Share map with recruiter/HM; redact if exporting externally.
