## Intake checklist

Gather from hiring manager, recruiter, and JD before sourcing:

| Field | Questions |
|---|---|
| Role | Title, level, team, reporting line |
| Scope | Top 3 outcomes in first 6–12 months |
| Skills | Must-have tech/domain; nice-to-have |
| Location | Onsite/hybrid/remote; time zones; relocation |
| Comp | Band or range; equity; visa sponsorship |
| Process | Interview stages, timeline, headcount |
| Slate | Diversity goals; internal transfer priority |
| Competition | Target companies; no-poach or partner constraints |

Document ambiguities as **assumptions** until confirmed.

## Ideal candidate profile

Structure ICP as searchable fields:

```
Titles:     [e.g., Staff SWE, Principal IC]
Seniority:  [years, scope: team vs org]
Skills:     [required keywords + synonyms]
Companies:  [tier 1 targets, acceptable, avoid]
Trajectory: [promotion pattern, scope growth]
Signals:    [OSS, talks, patents, certifications]
Anti-patterns: [job hop <12mo x3, wrong domain]
```

Map each ICP element to at least one **channel + filter**.

## Search strategy canvas

| Dimension | Decision |
|---|---|
| Pool size | Estimated addressable population |
| Urgency | Sprint (1–2 wk) vs evergreen |
| Volume | Outreach/week vs quality-first |
| Warmth | Referral-first vs cold-only |
| Geo | Expand radius if pool < N |
| Level | Up-level or down-level if bar blocks volume |

Set **weekly targets**: profiles reviewed, qualified adds, outreach sent, replies.

## Channel selection

| Persona | Primary | Secondary |
|---|---|---|
| Software IC | GitHub, LinkedIn, community Slack | Conference agendas, OSS contributors |
| Data/ML | Papers, Kaggle, LinkedIn | Meetup speakers |
| GTM | LinkedIn, alumni, CRM exports | Industry lists (licensed) |
| Executive | LinkedIn, board/advisor graphs | Referral-only often |
| Niche craft | Guilds, certifications, portfolios | Niche job boards |

Align channel effort with where **ICP density** is highest.

## Exclusions and knock-outs

Document hard excludes to save review time:

- Wrong level or function
- Geographic ineligibility
- Competitors under NDA/no-poach
- Recent outbound by another req (cooling period)
- Former employee on do-not-rehire list (recruiter-owned)

Soft flags (review with recruiter): short tenure, title inflation, gap years.
