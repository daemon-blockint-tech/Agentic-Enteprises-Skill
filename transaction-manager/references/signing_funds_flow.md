# Signing and Funds Flow

## Signing logistics

### Pre-signing checklist

- [ ] Final versions match agreed blackline
- [ ] Signature blocks match incumbency and DOA
- [ ] Exhibits and schedules attached and numbered
- [ ] CPs for signing (if any) documented
- [ ] Counterpart execution order agreed

### Methods

| Method | Notes |
|---|---|
| Wet ink | Originals for some jurisdictions |
| PDF / e-sign | Platform and audit trail |
| Closing call | Read names and versions aloud |

### Conformed copies

When parties sign separately, assemble **conformed** set with all signatures; distribute to data room and counsel.

## Funds flow memorandum

Document in one **FFM** (internal, counsel-reviewed):

| Line | Detail |
|---|---|
| **Purchase price** | Base, adjustments, currency |
| **Sources** | Equity, debt, rollover |
| **Uses** | Seller proceeds, escrow, fees, payoffs |
| **Payees** | Legal name, bank, account, reference |
| **Timing** | Simultaneous with closing vs escrow release |

### Escrow

- Amount, duration, release triggers
- Agent contact and wire instructions
- Dispute resolution mechanism

### Payoffs

- Lender payoff letters with per diem
- UCC releases and lien terminations tracked in matrix

## Simultaneous exchange

Typical order on closing call:

1. Confirm CPs satisfied or waived
2. Confirm regulatory and third-party items
3. Release signatures (if held in escrow)
4. Confirm wires initiated / received
5. File charter or other post-close filings per checklist

**No wire** until authorized signatory and counsel confirm CP clearance.

## Post-signing

- Distribute executed PDFs within agreed hours
- Update matrix to executed
- Archive in deal folder and data room
- Trigger post-close notifications (`communication-lead` for external)
